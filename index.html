# GitHub Pages — Ready‑to‑Deploy Starter Kit This kit gives you a small website that will deploy for **free** on GitHub Pages. It includes: * `index.html` — your homepage (with a YouTube embed placeholder) * `styles.css` — basic styling * `README.md` — one‑page instructions * `.nojekyll` — ensures GitHub Pages serves files as‑is * `404.html` — SPA‑style redirect (optional) * `CNAME` — (optional) add your custom domain --- ## 1) Create the repository 1. Go to GitHub → **New repository**. 2. Name it anything (e.g., `my-site`). Keep it **Public**. 3. Add **no** starter files here (we’ll paste the ones below). > OR: if you name it exactly `USERNAME.github.io` (replace `USERNAME` with your GitHub username), it becomes your special user site. --- ## 2) Enable GitHub Pages 1. Open **Settings → Pages**. 2. Under **Build and deployment → Source**, choose **Deploy from a branch**. 3. Set **Branch** to `main` and **/ (root)\`**. 4. Click **Save**. GitHub will deploy after you push files. --- ## 3) Add these files to the repo Create files with the exact names and content below. ### `index.html` ```html My Free Site
